# VRE Screening ML

This repository contains analysis code for the study  
"Development of machine-learning models for targeted PCR-based active surveillance of vancomycin-resistant Enterococcus using patient- and ward-level information”.

## Data availability

## Code availability
The code used for the analysis is available in the `src` and `notebooks` folders.

## Environment
Main libraries:
- Python
- scikit-learn
- XGBoost
- LightGBM
- CatBoost
- SHAP
- imbalanced-learn
